 Project Overview

**CampusPromoHub** is a visually engaging and conversion-focused landing page designed for a campus-based marketing agency.

The goal of this project was to create a **premium, agency-level website** that:

* Captures attention instantly
* Communicates services clearly
* Drives user action (conversion-focused design)

This project highlights strong skills in **UI/UX design, frontend development, and visual storytelling**.


 Key Features

🔹 **Modern UI/UX Design**
Clean layout, bold typography, and gradient aesthetics

🔹 **Interactive Experience**
Custom cursor, hover effects, and scroll animations

🔹 **Conversion-Focused Sections**

* Hero with metrics
* Services breakdown
* Portfolio showcase
* Pricing plans
* Testimonials

🔹 **Fully Responsive**
Optimized for desktop, tablet, and mobile devices

🔹 **Smooth Performance**
Lightweight and fast with pure HTML, CSS, and JS
